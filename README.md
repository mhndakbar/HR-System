# Amazon Clone 
> This project is part of Sprints Backend bootcamp.

## Tools :-
- RoR
- Mailhog
- Active storage for uploads
- Devise for authentication 

## Functions :-
- Send welcome email to each new user.
- Upload profile pic
- User authentication
- Show teams details
- Arabic translation
- Top and side navigators with footer
- Tables to list down items
- Complex forms
